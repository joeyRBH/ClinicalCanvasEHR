## Domain Availability Checking Instructions

### How to Check Domain Availability:

**Option 1: Quick Check (Free)**
1. Visit **Namecheap.com** or **GoDaddy.com**
2. Use their domain search tool
3. Enter domain name (e.g., "therapycore.com")
4. Instant availability results

**Option 2: Bulk Check**
1. Visit **InstantDomainSearch.com**
2. Type base name (e.g., "therapycore")
3. See availability across all TLDs at once

**Option 3: WHOIS Lookup**
1. Visit **who.is** or **whois.domaintools.com**
2. Look up specific domain
3. See registration details, expiration dates
4. Contact owner if domain is registered but unused

## Trademark Considerations

### Before Finalizing Name:

1. **USPTO Search** (Free)
   - Visit: https://www.uspto.gov/trademarks
   - Search trademark database
   - Check for conflicts in healthcare/software classes

2. **Google Search**
   - Search exact brand name in quotes
   - Check for existing usage in healthcare
   - Look for competing products

3. **Healthcare Directories**
   - Search Psychology Today
   - Check healthcare provider directories
   - Verify no existing EHR with same name

4. **App Stores**
   - Search iOS App Store
   - Search Google Play Store
   - Check for naming conflicts

### If Clear - File Trademark:
- Cost: $250-350 (USPTO filing fee)
- Attorney: $500-2,000 (optional but recommended)
- Processing: 8-12 months
- Classes: Software (Class 9), Healthcare Services (Class 44)
